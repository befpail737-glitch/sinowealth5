# Sinowealth Distributor Website

This is a professional distributor website for Sinowealth electronic components, developed as a complete static site with SEO optimization and technical support resources.

## Project Overview

**Title**: Sinowealth Distributor - Authorized Sinowealth Agent | Litong  
**Domain**: sinowealth.elec-distributor.com  
**Purpose**: Professional website for distributing Sinowealth 8-bit/32-bit MCU, BMS chips, OLED display drivers and providing technical support to customers globally.

## Features

- **Responsive Design**: Mobile-first approach with full responsiveness across all devices
- **SEO Optimized**: Optimized for search engines with meta tags, structured data, and semantic HTML
- **Lightweight**: Pure HTML/CSS/JavaScript with no external dependencies
- **Fast Loading**: Optimized performance for quick page loads
- **Accessible**: Implements accessibility best practices for users with special needs
- **Multi-language Ready**: Designed with internationalization in mind

## Deployment Instructions

1. The site is built with pure HTML/CSS/JavaScript with no build tools required
2. Deploy to any static hosting platform
3. For Cloudflare Pages deployment, it will automatically build and deploy
4. Ensure the following headers are set:
   - Cache headers for static assets
   - Security headers (X-Frame-Options, X-Content-Type-Options, etc.)
