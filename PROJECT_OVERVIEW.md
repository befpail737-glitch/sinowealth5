# Sinowealth Distributor Website

## Project Overview
This is a complete static website for a Sinowealth (矽旺) authorized distributor. Sinowealth is a leading Chinese semiconductor company specializing in 8-bit/32-bit MCUs, BMS chips, and display drivers. This website serves as a professional platform to market and distribute their products globally.

## Domain
**sinowealth.elec-distributor.com**

## Website Purpose
- Distribute Sinowealth 8-bit/32-bit MCU products
- Distribute BMS chips (especially SH36700 series for e-bike market)
- Distribute OLED/AMOLED display driver chips
- Provide technical support and resources
- Serve global customers requiring Sinowealth components

## Key Products Featured
1. **8-bit MCUs** - SH79F series for home appliance control
2. **32-bit MCUs** - ARM Cortex-M0 based controllers
3. **BMS Chips** - SH36700 series for e-bike and power tool applications
4. **Display Drivers** - OLED/AMOLED display controllers
